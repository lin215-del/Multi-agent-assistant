# 暨大学生助手 · 前端 UI 设计 Prompt

> 给前端 AI 工具（v0 / Lovable / Builder.io 等）使用的一份产品需求文档。
> 完成后输出三个文件：`index.html` + `style.css` + `app.js`（纯静态，FastAPI StaticFiles 托管）。

---

## 一、产品定位

**暨南大学学生事务智能问答助手**。学生用自然语言问教务问题（"怎么选课""奖学金怎么申请""挂科咋办"），系统检索后给出**带来源引用、可点开核对**的答案，检索不到时**拒答**而非胡编。

**差异化亮点（答辫主打）**：把 PDF 里的**表格**、**图**、**正文段落**三种素材分开检索，前端答案旁能区分"这一段引用自表格""这一段是图描述"。

**目标用户**：暨南大学本科生。**演示场景**：答辩现场，老师问问题→前端流畅回答→答案带类型徽章。

---

## 二、技术约束（必读）

1. **纯静态**：3 个文件 `index.html` + `style.css` + `app.js`，无 npm/Vite/React
2. **被 FastAPI 托管**：所有资源用相对路径，不写 `<script src="https://...">`
3. **hash 路由**：`#/chat` `#/cards` `#/history`，刷新页面路由保留
4. **现代浏览器**：Chrome / Edge，桌面分辨率 ≥ 1024px
5. **字体**：`"Microsoft YaHei", "Segoe UI", Arial, sans-serif`（中文环境）
6. **不引第三方库**（不 jQuery、不 Vue、不 React），原生 JS + fetch + DOM API

## 三、后端 API（务必完整对接）

基础路径：`http://localhost:8090`（开发）/ 答辩机同源。

### 1. `POST /api/chat`

**请求**：
```json
{ "question": "国奖申请条件是什么" }
```

**响应 200**：
```json
{
  "answer": "国家奖学金申请条件如下：\n1. 学习成绩优异，…（来源：通知-857469）",
  "route": "retrieve",                      // "retrieve" | "tool" | "both" | "reject" | null
  "matches": [
    {
      "content": "| 条件 | 要求 |\n| --- | --- |\n| 成绩 | GPA≥3.5 |",
      "source": "通知-857469",
      "score": 0.85,
      "type": "table"                       // "table" | "figure" | "text" —— 前端按这个打徽章
    },
    {
      "content": "图片描述：流程图展示了…",
      "source": "PDF-yingzai",
      "score": 0.78,
      "type": "figure"
    }
  ],
  "tool_output": null,                      // tool/both 时是字符串，retrieve/reject 时为 null
  "analysis": "草稿…（反思重试前的版本）",
  "reflection": { "ok": true, "reason": "引用了来源" },
  "round": 0,
  "latency_ms": 2300,
  "history_id": 42
}
```

**响应 422**：`{"detail": "question 不能为空"}`

### 2. `GET /api/cards`

返回服务卡片清单（手写 12 张）：

```json
{
  "groups": [
    {
      "name": "教务通知",
      "cards": [
        {
          "id": "通知-857469",
          "title": "暨南大学关于 2026 年秋季本科微专业招生报名的通知",
          "desc": "微专业报名时间、对象、流程；多个微专业集中招生。",
          "sample_q": "微专业怎么报名？",
          "source_doc": "通知-857469"
        }
      ]
    },
    {
      "name": "学生办事指南",
      "cards": [ /* 9 篇 */ ]
    }
  ]
}
```

### 3. `GET /api/cards/{id}`

返回单卡片（找不到 404，结构同上卡片）。

### 4. `GET /api/history?limit=50`

返回倒序历史列表 `[{id, question, route, answer, matches, reflection, analysis, round, latency_ms, created_at}, ...]`。

### 5. `GET /api/history/{id}`

返回单条完整 trace（同上卡片结构）。

### 6. `GET /api/documents`

返回 RAGFlow 知识库文档列表：

```json
{
  "docs": [
    { "name": "通知-857469.md", "chunk_count": 5, "run": "DONE" },
    { "name": "PDF-yingzai.md", "chunk_count": 12, "run": "DONE" }
  ],
  "count": 29
}
```

响应 503：`{"detail": "RAGFlow 不可用: ..."}`

## 四、三个页面（核心）

### 页面 1：问答页 `#/chat`（**答辫核心**）

**布局**（从上到下）：

```
┌─────────────────────────────────────────────────────────┐
│ 顶部导航：[问答] [卡片] [历史]              标题：暨大学生助手  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 输入你的问题…                              [问] │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ── 答案区（提交后出现）──                                │
│                                                         │
│  Q: 国奖申请条件是什么                                   │
│                                                         │
│  A: 国家奖学金申请条件如下：                              │
│     1. 学习成绩优异（来源：通知-857469）                  │
│     2. 综合素质突出（来源：通知-857469）                  │
│                                                         │
│  ▼ 引用来源（3 条，可折叠）                               │
│  ┌─────────────────────────────────────────────────┐   │
│  │ [表格] 通知-857469 · 相似度 0.85         [展开] │   │
│  │ ┌─────────────────────────────────────────────┐ │   │
│  │ │ | 条件 | 要求 |                             │ │   │
│  │ │ | --- | --- |                             │ │   │
│  │ │ | 成绩 | GPA≥3.5 |                         │ │   │
│  │ └─────────────────────────────────────────────┘ │   │
│  ├─────────────────────────────────────────────────┤   │
│  │ [图] PDF-yingzai · 相似度 0.78         [展开]   │   │
│  ├─────────────────────────────────────────────────┤   │
│  │ [正文] 通知-859020 · 相似度 0.65         [展开] │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  ── 过程 trace（debug 用，可折叠）──                       │
│  Route: retrieve · Round: 0 · Latency: 2.3s              │
│  Reflection: ✓ 引用了来源                                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**交互细节**：

- **输入框**：占位文字 "试试问：怎么选课 / 算 GPA / 国奖条件"
- **发送时**：禁用按钮 + 按钮文字变 spinner "思考中…"
- **拒答文案**：route="reject" 时答案区显示淡红色横幅 + "服务范围外"提示
- **工具结果**：tool_output 不为空时单独显示 "📊 工具计算：…"区块
- **反思重试**：round > 0 时显示 "⚠ 第一次答案不满意，自动重写 N 次"
- **每条 match 默认折叠**：点击行展开看 content 全文（表格保留 Markdown 格式）

**类型徽章颜色**（多模态差异化钩子，**必须按这个调**）：

| type | 徽章文字 | 背景色 | 文字色 |
|---|---|---|---|
| `table` | 表格 | `#dbeafe` 浅蓝 | `#1e40af` 深蓝 |
| `figure` | 图 | `#f3e8ff` 浅紫 | `#6b21a8` 深紫 |
| `text` | 正文 | `#f3f4f6` 浅灰 | `#374151` 深灰 |

## 五、视觉风格指引

**主色调**：暨南大学校色系——绿色 `#0f6f64`（主品牌）/ `#09544b`（深色变体）。

**辅助色**：
- 文字主色 `#172033`、次要 `#667085`
- 背景 `#f7f8fa`、卡片 `#ffffff`、分隔线 `#d9e0ea`
- 警告红 `#b42318`、成功绿 `#ecfdf3`

**形状**：
- 圆角 6-8px（按钮 / 输入框）/ 10-12px（卡片）
- 阴影：`0 1px 3px rgba(15,23,42,.06)` 浅层

**排版**：
- 正文 16px / 答案 18px / 标题 20px
- 行高 1.7（中文友好）
- 中文优先字体：Microsoft YaHei，回退 PingFang SC / sans-serif

**整体气质**：简洁专业、学术感、留白多，不要花哨动画。**答辩现场投屏要看得清**，避免低对比度小字。
