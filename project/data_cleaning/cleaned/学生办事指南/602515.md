# 打印中英文成绩单、证明等

> 来源：https://jwc.jnu.edu.cn/2021/0317/c11805a602515/page.htm

在校本科学生：
在校学生享有5份免费成绩单（其中1份为中文成绩单，由教务处统一打印，学生毕业时与证书同时领取。其他4份为2份中文+2份英文，学生自主安排打印），2份中英文在学证明。学生需要时，可登陆“暨南大学全日制本科学生自助打印系统”（各校区均有该设备。登陆账号为学号，密码为身份证后6位字符）选择打印成绩单或在学证明，预览无误后点击打印。系统首先打印免费份额，免费份额用完后，可用手机扫码付费打印。
打印英文成绩单和在学证明的学生应先要查看是否有英文名；英文名格式：内招生为姓名拼音（格式：姓和名分开写，姓和名的第一个字母大写，姓在前、名在后，例：李小山Li Xiaoshan）。外招生为有效身份证件上的英文名。英文名的填写请按照教务处通知时间要求填写。填写路径：教务处网页“新教务系统——搜索“学生信息变更”——申请变更——勾选“基本信息”——填写“变更后内容”，并上传附件（附件为有英文名的有效身份证件.jpg）——“申请变更提交”即可。（每位同学只能填写一次）。英文名有误的，请填写《普通申请表》（本科生院网页——下载服务——学籍相关文件），注明正确英文名，并附有效身份证件复印件，交到学院教科办，由学院统一交教务处学籍科修改。
打印在学证明要求学生先在教务管理系统完成注册，为在校状态。
已毕业学生：
一、成绩单打印：
2006年后入学的学生需要打印本科成绩单，可直接登陆“暨南大学全日制本科学生自助打印系统”（打印机在本部行政办公楼129门口。登陆账号为学号，密码为身份证后6位字符），预览无误后用手机扫码缴费打印。
2000-2006年入学的学生，请先电话咨询020-85220035。
2000级以前的毕业生需要成绩单，请咨询学校档案馆（电话020-85220237，蔡冠深博物馆四楼）。
二、英文毕业证明的办理：
2006年及以后入学的学生需要打印英文毕业证明，可直接登陆“暨南大学全日制本科学生自助打印系统”（打印机在本部行政办公楼129门口。登陆账号为学号，密码为身份证后6位字符），预览无误后用手机扫码缴费打印。
2006年以前入学的学生打印英文毕业证明，请带齐身份证、毕业证、学位证三个证件的复印件到教务处学籍科申请缴费办理。
已毕业学生打印英文成绩单和英文毕业证明时，如发现无英文名，请提供有效身份证件复印件，向学籍科提出协助处理申请。
成绩单、证明等文件的密封
学生在自助打印系统打印的成绩单、在学证明、英文毕业证明，需要教务处密封的，可到各校区密封办理点申请教务处专用信封密封、盖章。
（校本部在行政办公楼129室学籍科，番禺校区在番禺校区教学办、华文校区在华文学院教学办、珠海校区在珠海校区教学办、深圳校区在深圳校区教学办。）
缴费标准、缴费方式
缴费标准：根据粤价[2007]186号文规定的标准缴费(中文成绩单20元/份、英文成绩单40元/份、成绩单复印件加盖审核章和原印10元/份、中英文在学证明10元/份、英文毕业证明40元/份)。
缴费方式：手机扫码。
