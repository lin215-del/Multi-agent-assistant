# 全日制本科学生学位证书遗失补办

> 来源：https://jwc.jnu.edu.cn/2021/0317/c11805a602511/page.htm

根据国家教育部相关规定，学位证书原件仅提供一份，如若遗失，只能开具学位证明书。学位证明书仅提供一份，遗失无法补办。
暨南大学全日制本科毕业生学士学位证明书申请程序及注意要点如下：
一、受理对象
1994年及以后毕业并获学士学位的全日制本科学生；
二、办理地点及时间
教务处学籍科，咨询电话020-85227693；
受理时间为每年3月-6月，9月-12月（请提前进行电话咨询并预约）。
三、办理流程
申请人需在办理时间内，将完整无误的申请材料交至学籍科。学籍科在收到申请人材料后的10个工作日内通知申请人领取学位证明书（特殊繁忙工作时段可能会延迟），申请人凭身份证原件由本人签收领取；
四、申请办理须提供以下材料
1、在学校档案馆复印的录取名册、学籍表、毕业与学位审批表、证书签收册各1份（档案馆地址：校本部蔡冠深博物馆4楼，联系电话：020-85220237）；
2、《暨南大学全日制本科毕业生学士学位证明书申请表》（由学籍科现场发放，申请人亲笔填写）
3、彩色大1寸蓝底近照2张；
（注意事项：申请人必须提供包含学位证书编号的材料，否则无法办理学位证明书）
