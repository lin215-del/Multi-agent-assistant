# 暨南大学关于开展2026年新时代高校哲学社会科学原创性教材建设培育项目校内遴选工作的通知

> 来源：https://jwc.jnu.edu.cn/2026/0707/c6765a859122/page.htm

各教学单位：
根据《教育部办公厅关于开展2026年新时代高校哲学社会科学原创性教材建设培育有关工作的通知》（教材厅函〔2026〕7号）要求，学校决定启动2026年新时代高校哲学社会科学原创性教材建设培育（以下简称培育项目）校内遴选工作，现将有关事项通知如下：
一、建设要求
坚持主题主线。以习近平新时代中国特色社会主义思想为指导，把习近平经济思想、法治思想、文化思想、生态文明思想、强军思想、外交思想、党建思想及习近平总书记系列重要讲话、指示批示精神全面贯穿、有机融入哲学社会科学各学科专业知识。
坚持反映实践。充分反映中国共产党成立105年来、新中国成立70多年来的成就经验，以新时代10多年来伟大实践成就教育学生，坚定“四个自信”。汲取中华优秀传统文化丰厚滋养，挖掘其时代价值和精神力量并做好现代转化，吸收借鉴人类优秀文明成果。
坚持突出原创。以解答中国问题、阐释中国实践为出发点，注重知识创新、理论创新、方法创新，努力用中国道理总结中国经验，把中国经验提升为中国理论，推动原创性实践向原创性理论、原创性教材转化。
坚持适教利学。适应新时代大学生的学习习惯和认知特点，创新编写方式和呈现形式，提高针对性实效性。善讲故事、善用案例，注重鲜活性生动性。力求表述准确、语言精练，通俗易懂、篇幅适当。
坚持成果导向。采用自下而上与自上而下相结合的方式打造多样态成果，充分调动学校、学者的积极性主动性。盘活盘优已有教学、科研和教材建设成果，优化编审流程、提高编审效率，加大力度、加快进度，同步建设、尽快推出。
二、成果形式及建设方式
（一）成果形式
培育项目成果形式共5种样态，分别是：活页、案例、大纲、讲义、教程，供教师课堂教学和学生自主学习使用。
1.活页。结合教学需要，概要解析教学重点难点等，重在讲“点”，适合作为教学辅助资料使用。每种活页不少于10个要点，总篇幅5万字左右。每个要点一般包括：背景、概念、内涵、评析、教学应用等。
2.案例。聚焦教学主题，分析典型案例，重在评“例”，适合作为教学辅助资料使用。每种案例不少于10个典型案例，总篇幅5万字左右。每个案例一般包括：主题、背景、案例详情、评析、教学应用等。
3.大纲。提纲挈领地概述一门课程的基本结构和教学要点，重在论“纲”，适合作为教学辅助资料使用。篇幅5万字左右。大纲一般包括：课程简介、教学目标、教学内容、重点难点、考核要求、拓展阅读等。
4.讲义。以专题形式系统阐释和呈现一门课程的基本教学内容，适合作为主要教学资料使用。篇幅15—20万字左右。讲义一般包括：专题内容、拓展阅读、思考题、数字化资源等。
5.教程。分章节形式编排，完整呈现一门课程的逻辑结构和教学内容，综合运用文本、图表、音视频等生动呈现，具有一定系统性条理性，适合作为主要教学资料使用。篇幅20—25万字左右。教程一般包括：章节内容、思考题、参考文献、数字化资源等。
（二）建设方式
1.集中申报、择优培育。采取自愿申报的方式建设，鼓励跨院系、跨学科、跨领域联合攻关，支持有热情、有能力，能投入充足时间精力的高水平专家领衔。
2.编修结合、迭代升级。鼓励探索创新，主要依托现有学校组织编写、教师自编的教材、讲义、专著或重要教学、科研成果改编，进一步提高整体质量。
3.审核认定、立项支持。先审核、后认定。通过初审的由所在高校培育、按程序提交复审。通过复审的正式立项，作为“新时代高校哲学社会科学原创性教材建设培育成果”推出，同步纳入教育部哲学社会科学研究项目“高校哲学社会科学教材建设研究专项”；未通过的不予立项。
4.试教试用、持续建设。培育成果须试教试用，根据试教试用情况持续完善。建设方向和整体质量达到原创性教材要求并按程序送审，通过审核的纳入“新时代系列”教材。
三、申报范围及条件
（一）申报范围
分两类申报。自选项目重点建设哲学社会科学各学科专业基础课程、核心课程或部分公共基础课程对应的培育成果。其中，高校思政课教材、已建设的“中国系列”“新时代系列”等中国原创性哲学社会科学教材和马工程教材对应课程不再建设培育项目。补充建设部分急需紧缺项目（项目一览表见附件1）。学校优先推荐已立项的校级哲学社会科学原创性教材建设培育项目。
（二）申报条件
1.课程和成果。对应课程自2022年秋季学期以来须开设不少于1学期，不少于1学分。已有成果须基本符合建设要求，已形成教材、讲义、专著或重要教学、科研成果且与所申报项目内容高度关联。
2.单位和团队。申报单位须有健全的教材工作机制，具备组织编写、审核、试用、培训等条件，能够提供培育项目所需的人力物力和评价激励等支持保障，按时保质完成任务。项目负责人须政治强、业务精、作风正，年龄不超过65岁，熟知培育项目建设要求，具有较强的统稿能力。项目团队应分工明确、结构合理，总人数10人以内（含项目负责人），团队成员需为教学科研骨干。项目负责人及成员均不得跨组申报。
四、申报流程
个人申报。项目负责人填写《2026年新时代高校哲学社会科学原创性教材建设培育项目申报书》（附件2），并附上完整书稿及开课证明（见申报书附件清单）。
学院推荐。学院按照推荐条件审核材料，出具综合考察意见（附件3），本校教师可填在同一张表上，团队中非本校教师需由其所在单位一级党组织出具。学院填写《2026年新时代高校哲学社会科学原创性教材建设培育项目申报汇总表》（附件4）。于7月13日（周一）下午17时前将以上所有材料纸质版盖章后报本科生院教务处，电子可编辑版和PDF版发送至ojwjyk@jnu.edu.cn。
学校将择优遴选推荐项目，后续系统填报等事项另行通知。
