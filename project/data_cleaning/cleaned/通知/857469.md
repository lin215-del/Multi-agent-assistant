# 暨南大学关于2026年秋季本科微专业 招生报名的通知

> 来源：https://jwc.jnu.edu.cn/2026/0621/c6765a857469/page.htm

各学院及全日制本科学生：
学校将于2026年秋季学期开设一批本科微专业，现将报名具体事项通知如下：
一、本科微专业介绍
本科微专业是在学生主修专业之外，依托学校优势学科，围绕特定方向开设的课程组。通过微专业学习，拓宽学生的专业视野、培养跨学科综合素质、帮助学生补齐知识和技能结构短板，实现复合型人才培养目标，提升学生的就业竞争力。2026—2027学年共有18个微专业面向学生招生，具体微专业简介、招生计划、报名要求见附件1。
二、报名条件及要求
（一）2023级、2024级、2025级全日制本科学生（六年制医学专业可招收2022级学生），在确保完成主修专业且学有余力的基础上，并且没有正在修读的辅修学士学位或辅修专业的学生均可报名，须同时符合各微专业开设的有关要求（见附件1）。
（二）每位学生限选报一个微专业。
三、报名时间及流程
请有意向修读的学生提前了解各微专业开设信息（见附件1），并按规定时间在本科教务管理系统（网站链接：https://jw.jnu.edu.cn/new/index.html）中提交申请（操作指南见附件2）。如有疑问，请咨询微专业所在学院。
（一）报名时间
6月23日8:00至7月3日17:00。截至6月29日17:00，对报名人数不足30人的微专业，学校统一予以撤班。如报名的微专业因人数不足撤班，学生可在报名时间内选报其他微专业，请学生及时留意报名状态。
（二）考核及录取
各微专业开设单位须提前制定严谨、规范的选拔程序与考核办法，确保学生遴选工作公平、公正、公开。录取结果应于7月12日前在学院网站予以公示，公示期不少于三个工作日。学院须于公示当天将公示链接及截图发送至教务处学籍科邮箱ojwxjk@jnu.edu.cn。公示无异议后，学院于7月16日前在本科教务管理系统微专业模块中完成招生审核。审核通过的学生名单须加盖单位公章，报送至教务处学籍科（行政办公楼129室）备案。学生可在7月17日登录本科教务管理系统查询最终录取结果。
四、培养要求及注意事项
（一）此次微专业学制为一年，不收取费用。学校将持续面向全日制本科学生开设微专业。因上课时间冲突，正在修读辅修学士学位、辅修专业的学生可完成修读后再报名后续微专业。
（二）成功录取的学生请按时参加微专业学习，计划自9月开学后逢周末线下授课，具体由各微专业所在学院通知。
（三）微专业学分不计入主修专业总学分，已完成的微专业课程可出具微专业课程成绩单。完成微专业修读要求的学生可获微专业学习证明，微专业不具有学士学位授予资格。
（四）微专业在读学生如申请退读，应向微专业所在学院提出，并在系统中提交申请。无正当理由退读学生，将取消后续所有年度的微专业报名资格。
五、咨询与联系
各微专业均设置QQ群等联系咨询方式（详见附件1），学生可通过QQ详细了解各微专业信息，报名期间如遇问题请及时咨询微专业所在学院负责老师。
