# RAGFlow 导入说明

## 建议知识库

名称：`暨南大学学生助手`

描述：面向暨南大学学生，提供常用表格、办事指南、学籍流程、新生材料、学生事务政策等查询。

## 上传内容

1. 上传 `data/cleaned/ragflow_markdown` 下的 Markdown 文件。
2. 上传 `data/files` 下的原始附件，比如 docx、pdf、xlsx。
3. 对表格/模板类文件，保留原附件比只保留抽取文本更重要，因为学生最终需要下载原模板。

## 回答要求

建议在智能体提示词里加入：

```text
你是暨南大学学生助手。回答时优先给出可办理事项、所需材料、适用对象、办理入口或附件名称。若知识库中有来源链接或附件链接，必须列出。无法确认时要说明“当前知识库未收录”。
```

## 第一批测试问题

- 本科生请假申请表在哪里下载？
- 新生想保留入学资格需要什么表？
- 学生证遗失怎么补办？
- 转专业申请表在哪里？
- 在校证明和成绩单怎么打印？
